# React + TypeScript + Vite + FSD

> Данный проект является учебный. 
> Пожалуйста, не ищите в нем практическое применение :)

## Стэк:

1. React
2. Mobx
3. Vite
4. PostCSS
5. Mock Server (MSW)

## Запуск

1. Установите зависимости

```bash
pnpm install
```

2. Запустите проект

```bash
pnpm start
```
