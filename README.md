# React + TypeScript + Vite + FSD

> Данный проект является учебный. 

## Цель разработки

1. Практика FSD
2. Практика Mobx
3. "Пощупать" Post CSS
4. Опробовать ZOD

## Стэк:

1. React
2. Mobx
3. Vite
4. PostCSS 
5. React Hook Form
6. Zod 
7. Mock Server (MSW)

## Запуск

1. Установите зависимости

```bash
pnpm install
```

2. Запустите проект

```bash
pnpm start
```
