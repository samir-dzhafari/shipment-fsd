# React + TypeScript + Vite + FSD

> Данный проект является учебный. 
> Пожалуйста, не ищите в нем практическое применение :)

## Цель разработки

1. Практика FSD
2. Практика Mobx
3. "Пощупать" Post CSS
4. Опробовать ZOD

## Стэк:

1. React
2. Mobx
3. Vite
4. PostCSS 
5. Mobx utils 
6. TanStack Form (React Hook Form)
7. Zod 
8. Mock Server (MSW)

## Запуск

1. Установите зависимости

```bash
pnpm install
```

2. Запустите проект

```bash
pnpm start
```
