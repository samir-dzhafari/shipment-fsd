# React + TypeScript + Vite + FSD

> В разработке!

> Сервис для распределения товара в грузовые автомобили.
> Данный проект является учебный и не имеет коммерческое применение.

## Стэк:

1. React
2. Mobx
3. Vite
4. PostCSS 
5. React Hook Form
6. Zod 
7. Mock Server (MSW)

## Запуск

1. Установите зависимости

```bash
pnpm install
```

2. Запустите проект

```bash
pnpm start
```
